# 💰 Personal Finance Tracker (Streamlit + AI)

A simple and intelligent personal finance tracking web app built using **Streamlit**, **SQLite**, and **Google Gemini AI**.  
It helps users track expenses, visualize spending, manage budgets, and get AI-powered financial advice.

---

## 🚀 Features

- 📝 Add daily expenses (date, category, amount, description)
- 📋 View all expense records in a table
- 📊 Dashboard with:
  - Total expenses
  - Total number of records
  - Top spending category
- 📈 Pie chart visualization of spending
- 💵 Monthly budget tracker with remaining balance
- 🗑 Delete expense by ID
- 🤖 AI Financial Advisor (Gemini AI):
  - Gives smart saving tips based on spending pattern

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🎈
- SQLite 🗄️
- Pandas 📊
- Matplotlib 📈
- Google Generative AI 🤖 (Gemini)
